# Coin

Coin é um projeto MOBILE onde o usuário encontra informações sobre CRIPTOMOEDAS.

## Tecnologias utilizadas:

- RN CLI
- Javascript
- Styled Components
- React Native
- Hook (useState | useEffect)
- FlatList
- Modal
- API CoinStats
- Push Notification com Firebase (FCM)
- Axios
